## 2.版本控制

1. 在git Bash中进入项目文件目录

2. 初始化当前目录

   ```shell
   git init	#初始化当前项目目录
   ```

3. 分析当前项目目录命令：

   ```shell
   git status 	#标红色文件名都是还未被管理的文件
   ```

4. 把文件添加到git管理列表中的代码：

   ```shell
   git add $.$ #添加指定文件$.$为你要添加的文件名
   git add .	#添加所有没有被管理的文件到git管理列表
   ```

5. 生成版本新的版本代码：

   ```shell
   git commit -m 'V1'	#生成新的版本'V1'可以写自己的版本描述
   ```
